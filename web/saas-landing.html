Provedor não configurado

Nesta instalação cada VPS corresponde a um provedor. Ainda não há provedor configurado nesta máquina.

Como criar o provedor — passo a passo

  1. 1
    Entrar na pasta do projeto (na VPS):
    cd /var/www/otyis-isp
  2. 2
    Rodar o instalador:
    sudo ./installer/install.sh
    A instalação fica nesta pasta. Quem usa PM2: sudo ./installer/install.sh --pm2
  3. 3
    Responder às perguntas do script: nome do provedor, slug, banco PostgreSQL, e-mail e senha do usuário Master, chave RADIUS (ou deixar em branco para gerar).
  4. 4
    Ao terminar, acesse o Painel do dono do sistema para ajustes ou o Portal com o usuário Master.
